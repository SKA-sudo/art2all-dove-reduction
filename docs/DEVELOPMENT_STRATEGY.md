################################################################
Art2all – Development Strategy
################################################################

Principle

Research exists to improve the product.

A validated hypothesis is only complete when it produces
a visible improvement of the Peace Dove.

------------------------------------------------------------

Reduction Laboratory

Purpose

Discover perception rules.

Output

Validated hypothesis.

------------------------------------------------------------

Perception Laboratory

Purpose

Apply validated hypotheses to the real Peace Dove.

Question

Does the visual result improve?

YES

→ integrate into Product Engine

NO

→ return to Reduction Laboratory

------------------------------------------------------------

Product

Only validated rules become part of the production engine.

------------------------------------------------------------

Git Workflow

dev

Research
↓

Perception Validation
↓

Visual Validation

↓

main

Only validated improvements.
