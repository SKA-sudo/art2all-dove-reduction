Leitgedanke
Art2all ist ein weltweites, generationsübergreifendes Kunstprojekt, das Menschen durch Kreativität verbindet.
Kinder aus allen Ländern malen, was Frieden für sie bedeutet. Jede Zeichnung wird zu einer Feder einer gemeinsamen Friedenstaube.
Die Friedenstaube ist kein statisches Kunstwerk.
Sie lebt.
Sie schläft.
Sie träumt.
Sie erwacht.
Sie fliegt.
Und sie lebt nur durch die Menschen, die ihr gemeinsam Leben schenken.
________________________________________
Die zentrale Botschaft
"Gemeinsam erwecken wir die Friedenstaube."
Nicht ein Künstler.
Nicht eine Nation.
Nicht eine Organisation.
Sondern Millionen kleiner Beiträge bilden gemeinsam etwas Größeres.
Jede Zeichnung zählt.
Jede Feder zählt.
Jedes Kind zählt.
________________________________________
Die Reise des Besuchers
1. Die Begegnung
Ein Besucher öffnet Art2all.
Er sieht eine wunderschöne Friedenstaube.
Nicht mehr.
________________________________________
2. Die Entdeckung
Beim Näherkommen erkennt er:
Die Taube besteht aus Federn.
________________________________________
3. Die Überraschung
Noch näher.
Die Federn sind gar keine Federn.
Es sind Kinderzeichnungen.
________________________________________
4. Die Geschichte
Ein Klick auf eine Zeichnung.
Sie wird groß.
Jetzt entdeckt der Besucher die Geschichte eines Kindes.
Nicht irgendein Bild.
Sondern einen Gedanken über Frieden.
________________________________________
Die Erzählung der Taube
Die Friedenstaube besitzt einen Lebenszyklus.
🌙 Sleeping
Die Taube ruht.
Sie wartet.
________________________________________
✨ Dreaming
Die ersten Zeichnungen erscheinen.
Sie beginnt zu träumen.
________________________________________
🌅 Awakening
Immer mehr Kinder schenken ihr Federn.
Sie beginnt zu erwachen.
________________________________________
🕊️ Awake
Sie lebt.
Sie bewegt sich.
Sie strahlt Hoffnung aus.
________________________________________
🌍 Flying
Die Welt hat sie gemeinsam erweckt.
Sie erhebt sich.
Nicht durch Software.
Sondern durch die Menschen.
________________________________________
Die Rolle der Kinder
Jede Zeichnung beantwortet dieselbe einfache Frage:
"Was bedeutet Frieden für dich?"
Die Kinder erschaffen nicht nur Bilder.
Sie schenken der Taube Leben.
Jede Zeichnung wird zu einer Feder.
________________________________________
Die Rolle der Länder
Art2all soll verbinden – nicht trennen.
Darum gibt es eine gemeinsame Weltgeschichte.
Jedes Land trägt dazu bei.
Länder dürfen sichtbar sein:
•	Anzahl der Beiträge 
•	Geschichten 
•	Kultur 
•	Projekte 
Aber niemals als Wettbewerb.
Die Botschaft lautet:
Alle Länder bauen gemeinsam an derselben Friedenstaube.
________________________________________
Was geschieht nach dem Erwachen?
Eine erwachte Friedenstaube verschwindet nicht.
Sie wird Teil eines lebendigen Himmels.
Eine neue schlafende Taube beginnt.
Dadurch entsteht ein Symbol:
Frieden ist niemals abgeschlossen.
Jede Generation erschafft ihre eigene Friedenstaube.
Und jede erwachte Taube erinnert die nächste Generation daran, dass Frieden immer wieder neu entstehen muss.
________________________________________
Technische Philosophie
Die Technik dient niemals sich selbst.
Sie unterstützt immer die Geschichte.
Darum gilt:
•	Papier soll wie Papier aussehen. 
•	Federn sollen wie Federn wirken. 
•	Licht soll Hoffnung vermitteln. 
•	Die Taube soll organisch aus Kinderzeichnungen wachsen. 
•	Von weitem eine Taube. 
•	Von nahem tausende Geschichten. 
________________________________________
Langfristige Vision
Art2all soll mehr sein als eine Website.
Es kann werden:
•	ein weltweites digitales Kunstwerk 
•	eine Bildungsplattform 
•	eine Friedensinitiative 
•	eine interaktive Ausstellung 
•	eine öffentliche Installation 
•	ein Symbol dafür, dass viele kleine Beiträge gemeinsam etwas Großes schaffen. 
________________________________________
Unser Leitsatz für jede zukünftige Entscheidung
Bei jeder neuen Idee fragen wir nicht:
"Ist das technisch interessant?"
Sondern:
"Macht diese Entscheidung die Geschichte stärker?"
Wenn sie den Moment unterstützt, in dem ein Besucher erkennt:
„Unglaublich … diese Friedenstaube lebt, weil Millionen Kinder ihr gemeinsam Leben geschenkt haben.“
…dann gehört sie zu Art2all.
