# Purpose
################################################################

This document collects validated observations about
human visual perception discovered during the
development of Art2all.

It is not an implementation document.

It is not an architecture document.

It is not a software specification.

It documents our current understanding of visual
perception.

Every statement contained in this document originates
from observation, experimentation or artistic
validation.

The purpose of this document is to guide future
development of the Perception Engine.

Project First.

Science Second.

The Peace Dove remains the benchmark.

Scientific understanding is a consequence of building
the project.

################################################################
# Current Theory
################################################################
################################################################
PP-001
################################################################

Human perception is hierarchical.

Visual information is not processed simultaneously.

Perception progresses through successive semantic
stages.

The current working hierarchy is

Attention

↓

Whole Dove

↓

Head

↓

Eye

↓

Beak

↓

Body

↓

Wing

↓

Tail

This hierarchy represents the current working
hypothesis and may evolve through future experiments.

################################################################
PP-003
################################################################

Semantic meaning emerges from relationships.

Individual polygons contain no semantic meaning.

Meaning emerges only through relationships between
semantic regions.

The engine therefore operates on semantic structures
rather than technical geometry.

################################################################
PP-004
################################################################

Every semantic object is represented by a semantic
region.

Never by a single polygon.

Regions generate reference points.

Reference points generate relationships.

Relationships generate semantic meaning.

################################################################
PP-005
################################################################

The Production Engine never invents perception.

The Perception Engine discovers perceptual rules.

Only experimentally validated rules become part of
the Production Engine.
