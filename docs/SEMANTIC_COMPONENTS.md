# Semantic Components

Status: Living Document

---

## Purpose

Semantic Components are the first semantic entities derived from geometric observations.

They are not geometry.

They are not recognition.

They are the building blocks from which higher perception emerges.

Pipeline:

Geometry
↓

Observation
↓

Semantic Component
↓

Relationship
↓

Organization
↓

Recognition
